# boids
Boids in C.

https://github.com/user-attachments/assets/bb738ca2-52d6-4f91-bbf3-9d6b01c8c1a8

## Usage
* Install `raylib`, compile, and add lib file to `boids/lib`
* Run `make` to compile, and `./boids` to run
